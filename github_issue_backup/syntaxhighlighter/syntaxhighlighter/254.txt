    "body": "what I had to do:\r\n  * install latest version of node/npm (package manager version on ubuntu was out of date, built mine from nodejs.org)\r\n  * install jake (as root, ```npm install -g jake```, couldn't get it to work without global install)\r\n  * install node-sass (```npm install node-sass```, might be a missed dependency?)\r\n  * install bower (also as root, ```npm install -g bower```, also wouldnt work without global install)\r\n  * run ```bower install``` (should give you a bower_components directory)\r\n  * edit build/build.coffee and tests/tests.coffee to rename 'components' to 'bower_components' (or maybe rename the dir, might be easier)\r\n  * finally run ```jake```\r\nand after all that, you get the output in the pkg/ directory!"
