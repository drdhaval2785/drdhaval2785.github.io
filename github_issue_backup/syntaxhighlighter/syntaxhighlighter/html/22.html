pixit commented on 2012-02-03 14:56:54

After searching a little bit, I found the solution to your problem, though it might be a bit late.

In scHore.css look for this text :

.syntaxhighlighter .line {
white-space: pre !important;
}

And replace it by

.syntaxhighlighter .line {
white-space: pre-wrap !important;
}
