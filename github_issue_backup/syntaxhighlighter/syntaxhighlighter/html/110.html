Is there a way you can have a different brush for each code block in the same HTML page?


toastedcrumpets commented on 2012-05-27 19:36:44

Yes, quite easily. Its covered in the documentation, you just need to set the brush: option to the block containing your code to highlight. E.g. For the pre interface you just do \<pre class="brush: js;"> .... \</pre> And you can pick js or cpp or bash or plain etc.
