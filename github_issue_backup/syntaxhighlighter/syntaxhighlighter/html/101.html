Ok, I just browsed through https://github.com/alexgorbatchev/SyntaxHighlighter/blob/master/scripts/shCore.js#L288 and using SyntaxHighlighter.highlight() does the trick for me.

At https://github.com/alexgorbatchev/SyntaxHighlighter/blob/master/scripts/shCore.js#L353 you can see that the function SyntaxHighlighter.all() is bound to the event load of the window object. The function call in line 358 brought me to the function mentioned above.
