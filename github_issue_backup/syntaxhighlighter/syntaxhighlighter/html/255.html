IE Problems - Last line not shown and copies all lines as single line

deadlydog commented on 2013-09-10 16:56:29

There seems to be some problems with SyntaxHighlighter and the latest version of Internet Explorer (v10):

  1. For any snippets long enough to require a vertical scroll bar, when you scroll to the very bottom it only scrolls down to the 2nd last line, so the last line is never visible.

  2. If you copy multiple lines from a snippet, they all get pasted as a single line with all of the line breaks removed.

For an example, take a look at the following page and the comments at the bottom describing the problem a little more: http://blog.danskingdom.com/powershell-ise-multiline-comment-and-uncomment-done-right-and-other-ise-gui-must-haves/

Everything still works fine in Firefox and Chrome.
