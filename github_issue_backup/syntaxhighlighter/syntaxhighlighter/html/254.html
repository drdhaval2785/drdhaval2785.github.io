ian-kent commented on 2013-09-24 21:03:34

what I had to do:

  • install latest version of node/npm (package manager version on ubuntu was out of date, built mine from nodejs.org)
  • install jake (as root, npm install -g jake, couldn't get it to work without global install)
  • install node-sass (npm install node-sass, might be a missed dependency?)
  • install bower (also as root, npm install -g bower, also wouldnt work without global install)
  • run bower install (should give you a bower_components directory)
  • edit build/build.coffee and tests/tests.coffee to rename 'components' to 'bower_components' (or maybe rename the dir, might be easier)
  • finally run jake and after all that, you get the output in the pkg/ directory!
