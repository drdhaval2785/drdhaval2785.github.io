JScript brush: keyword "finally" is missing

arimeq commented on 2012-07-10 15:14:09

JavaScript has a full try..catch..finally statement, like this:

try { // some commands that may raise an exception, like open file descriptor and read contents } catch (e) { // exception handling, like weird characters are read from what should be a text file } finally { // what to do after all commands, even if there was an exception, like closing a file descriptor }

JScript brush does not support last block. It is sufficient to add word "finally" at the end of line 25 in file scripts/shBrushJScript.js, in variable "keywords".
