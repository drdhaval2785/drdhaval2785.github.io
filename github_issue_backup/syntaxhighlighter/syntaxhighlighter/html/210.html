wallneradam commented on 2013-05-06 11:51:51

Hi, I've just found this problem as well. I've discovered that this is not a problem with the SyntaxHighlighter, but the browser. You need to encode HTML entities. All your "<" and ">" must be "&lt;" and "&gt;". If you use PHP to print the text simply use htmlentities($xmlText, ENT_COMPAT, 'UTF-8');
