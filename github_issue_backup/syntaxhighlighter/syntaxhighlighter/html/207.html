randomessence commented on 2013-04-29 12:56:59

You can do this:

XRegExp.install('natives');

For example i use it like this with xregexp 3:

<script type="text/javascript" src="path/to/xregexp-all.js"></script>
<script type="text/javascript">
// <![CDATA[
XRegExp.install('natives');
// ]]>
</script>
<script type="text/javascript" src="path/to/shCore.js"></script>
<link href="path/to/shCoreDefault.css" rel="stylesheet" type="text/css" />
<link href="path/to/shThemeDefault.css" rel="stylesheet" type="text/css" />

It is written here https://github.com/alexgorbatchev/SyntaxHighlighter/issues/127
