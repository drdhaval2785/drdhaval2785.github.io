alexgorbatchev commented on 2013-04-19 23:32:05

If you look at https://github.com/alexgorbatchev/SyntaxHighlighter/blob/master/scripts/shCore.js#L353 you can see that all() hooks into window load event and calling it after window loaded will do nothing. Call highlight() instead.
