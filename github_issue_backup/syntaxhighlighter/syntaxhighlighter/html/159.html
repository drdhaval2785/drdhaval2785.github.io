yamiko-ninja commented on 2012-10-31 04:56:15

I updated the function list to support all functions listed on http://php.net/quickref.php

Updated the constants and keywords listed on http://php.net/manual/en/reserved.keywords.php and http://www.php.net/manual/en/reserved.constants.php

Also updated the variable definition based on http://www.php.net/manual/en/language.variables.basics.php

PHP technically considers true, false, and null as constants however most IDEs would consider them keywords. Thus I decided to include it in the keywords and not the constants.
