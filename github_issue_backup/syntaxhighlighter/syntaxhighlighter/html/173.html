Comment in the middle

joydeep1982 commented on 2012-12-26 08:53:35

If the comment is starting from the middle of the line and ending before the end of the line then sysntax highlighter is not taking care of the same ...

Example

public void setGender(Integer iGender)
{
    this.json.put(JSonConstant.GENDER, iGender/*gender.getValue()*/);
}

public Integer getStatus()
{
    return this.json.optInt(JSonConstant.STATUS);
}

In this case syntax highlighter is commenting the entire file starting from the position of the comment. It is unable to track the end of the comment which is not at the end of the line. Please check the image attached with the issue.
