scootergrisen commented on 2012-06-19 23:21:29

A solution might be to add the folowing linie in the "styles/shCore.css" file :

overflow: hidden !important;

Like so :

.syntaxhighlighter a, .syntaxhighlighter div, .syntaxhighlighter code, .syntaxhighlighter table, .syntaxhighlighter table td, .syntaxhighlighter table tr, .syntaxhighlighter table tbody, .syntaxhighlighter table thead, .syntaxhighlighter table caption, .syntaxhighlighter textarea { ... ... ... overflow: hidden !important; }

Maby not perfect solution but at least i can get rid of the horisontal scrollbar in some browsers (Safari, Chrome).


retronym commented on 2013-09-30 09:07:51

I've run into the same (or similar) problem on this page: https://issues.scala-lang.org/browse/SI-7884 at 90% zoom in Chrome Mac.

I've had some luck using the following style to stop the vertical scrollbar appearing (which in turn prevents unwanted horizontal scrollbars.)

.syntaxhighlighter {
  overflow-y: hidden!important;
  overflow-x: auto!important;
  overflow: inherit;
}
