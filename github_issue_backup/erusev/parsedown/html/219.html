johanjanssens commented on 2014-09-08 21:41:41

@smanzi

  1. MIT license

The Expat License, also referred to as the MIT license is GPL compatible. This means you can distribute MIT licensed code along with GPL licensed code.

In your case, if you create a Joomla plugin or extension, your code will need to be licensed GPL, you can however make use of a MIT library like Parsedown and distribute it this along with your plugin or extension.

  1. JED rules

The page you are linking to defies the license rules for the JED or Joomla Extension Directory. If you wish to publish and extension on the JED additional rules apply which are more strict. The JED doesn't accept extensions licensed MIT.

You can however still publish your extension as GPL which you are required to and distribute Parsedown along side it. This is not a problem at all and is allowed.

As Matthew writes here '...we won't list extensions that are completely MIT licensed...'

If you create an extension for Joomla and distribute Parsedown alongside it you are not creating an extension that is completely MIT. Your extension code, even if very simple would be GPL and make use of Parsedown which is MIT.

This is perfectly fine for the JED. JED rules only apply if you wish to put your extension on the JED. If you don't then no need to worry about them

Hope that helps.
