hkdobrev commented on 2014-06-16 19:41:13

BTW I've found the library GitHub uses for sanitization of readmes on github.com for a number of formats.

Here it is: https://github.com/github/markup

As you can see they perform sanitization on various markup languages: Markdown, Textile, plain text and others. So they do not expect the parser to sanitize its input/output.

Markdown in particular is allowing HTML so this makes sense.
