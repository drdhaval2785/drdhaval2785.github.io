URLs are converted to links

alehandrof commented on 2015-01-26 14:26:46

Given this input:

<a href="http://example.com">http://example.com</a>

Parsedown outputs:

<p><a href="http://example.com"><a href="http://example.com">http://example.com</a></a></p>

Clearly this is not desirable.

(P.S. I'm coming from the Statamic forums, where this issue was documented.)


erusev commented on 2015-01-26 14:37:26

You can prevent this by disabling automatic linking of URLs: $Parsedown->setUrlsLinked(false).
