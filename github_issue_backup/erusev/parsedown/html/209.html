CSS Link Stops Parsing

markfick commented on 2014-08-26 11:42:45

If you have a css link in body: 

<link rel="stylesheet" href="file.css">

Any markdown below it is not parsed.

(Even though link is not typically in body, it happens sometimes)


kminchev commented on 2014-08-26 15:10:35

Temporary workaround is to close the link element.
