Parsedown is currently not compatible with PHP 7

kelunik commented on 2015-02-01 23:51:11

PHP 7 doesn't allow this any longer: http://3v4l.org/62FEh

<?php

class Foo {
    private $a = "php7";

    function bar(array $data) {
        return $this->$data['prop'];
    }
}

$foo = new Foo;

var_dump($foo->bar(['prop' => 'a']));

All code using something like this has to be changed to:

<?php

class Foo {
    private $a = "php7";

    function bar(array $data) {
        return $this->{$data['prop']};
    }
}

$foo = new Foo;

var_dump($foo->bar(['prop' => 'a']));

e.g. here: https://github.com/erusev/parsedown/blob/master/Parsedown.php#L1405
