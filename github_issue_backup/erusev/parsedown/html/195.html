cebe commented on 2014-07-15 13:00:14

The parsing result is correct because you could have lazy lists like this:

- first item first line
first item second line
- second item first line
second item second line

All Markdown parsers do it the same way: http://johnmacfarlane.net/babelmark2/?normalize=1&text=-+first+item+first+line%0Afirst+item+second+line%0A-+second+item+first+line%0Asecond+item+second+line%0A

When you want to separate blocks you have to separate them by at least one blank line.
