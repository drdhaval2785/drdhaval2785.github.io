@hkdobrev Hi, I've seen your project before.

I just put a temp variable to log the break status, for example

// php code
for ($i = 0; $i < 100; $i ++) {
    for ($j = 0; $j < 50; $j ++) {
        if ($i * $j > 100) {
            break 2;
        }
    }
}
// javascript code
pass = false;
for (i = 0; i < 100; i ++) {
    for (j = 0; j < 50; j ++) {
        if (i * j > 100) {
            // log status and break immediately
            pass = true;
            break;
        }
    }

    // check the variable and break again
    if (pass) break;
}
