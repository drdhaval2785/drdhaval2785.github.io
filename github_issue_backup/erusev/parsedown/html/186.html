(http://pythonhosted.org/Markdown/extensions/attr_list.html) that lets you add html key/value pairs, and class/id labels. The syntax is for this is:

![A picture of a cat](cat.png){: style="float:right"}

Or, if embedded styling doesn't float your boat,

![A picture of a cat](cat.png){: .floatright}

with corresponding stylesheet stylish.css:

.floatright {
    float: right;
    /* etc. */
}
