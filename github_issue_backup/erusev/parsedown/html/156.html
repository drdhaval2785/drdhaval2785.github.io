Align tables with 'style' attribute instead with 'align'

kminchev commented on 2014-04-17 16:44:51

This is how Parsedown parses aligned tables:

| A1 | B1 | C1 |
| :- | - | - |
| a1 | b1 | c1 |

<td align="left">a1</td>

Since web and CSS are slowly moving toward HTML5, you should use 'style' attribute:

<td style="text-align: left;">a1</td>
