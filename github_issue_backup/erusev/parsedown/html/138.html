hkdobrev commented on 2014-03-27 11:42:15

@schucan From the Semantic Versioning Specification (SemVer):

Major version zero (0.y.z) is for initial development. Anything may change at any time. The public API should not be considered stable.

You should check for yourself whether parsedown is in a stable enough version for you. The API and the parsing may change drastically before reaching a 1.0 release.

However @erusev has provided correct minor (x.X.x) version bumps when anything major was added/changed so you are free to use any version which suits your needs and you could use it forever.

I hope your issues are resolved in the future. Cheers!
