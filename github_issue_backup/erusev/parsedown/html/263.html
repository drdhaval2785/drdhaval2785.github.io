<p><gallery

erusev commented on 2015-01-13 21:27:18

In order for Parsedown to recognise this as an HTML element, the opening tag should be on a single line.

I wish there could be an array of elements that should be considered block, similar to $voidElements and $textLevelElements(which I have no idea what are used for).

Parsedown considers all elements that are not text level to be block level.
