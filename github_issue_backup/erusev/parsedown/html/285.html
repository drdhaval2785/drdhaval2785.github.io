henriquemoody commented on 2015-02-09 17:27:44

@erusev, no it's not in the spec but If you take a look on all processed Markdown files rendered by GitHub you will he something like that right after <hN> tags:

<a name="this-is-an-anchor" class="anchor" href="#this-is-an-anchor">
    <span class="octicon octicon-link"></span>
</a>

I don't think put these tags on processed Markdown files is a good idea because it's based on the GitHub's CSS, but with the ids we still have the reference and we can link the titles. It's useful when you want to write a table of contents or just create an link to the same document.
