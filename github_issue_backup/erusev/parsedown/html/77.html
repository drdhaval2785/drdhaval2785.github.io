hkdobrev commented on 2014-02-05 00:33:57

@wkpark Yes, that's right. According to the way I understand the GFM spec, this is not a table, because it does not have a line with - separating the head from the body.

You can create tables by assembling a list of words and dividing them with hyphens - (for the first row), and then separating each column with a pipe |.
