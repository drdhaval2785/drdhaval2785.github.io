Allow user to set a relative path

mulquin commented on 2014-06-16 13:36:16

I wrote this for one a project where markdown files are strewn across multiple folders.

Inserting images was becoming troublesome as the full path for each image had to be set in each file.

This little work-around can be used to dynamically change the relative path to the folder with the markdown file inside.

Unsure if more appropriate for MarkdownExtra or not.
