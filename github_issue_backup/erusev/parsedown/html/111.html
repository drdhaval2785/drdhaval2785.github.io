erusev commented on 2014-04-24 21:28:23

I believe that the latest version resolves this.

In order to process code blocks, you should write an extension that defines completeCodeBlock.

The following example would convert texts of code blocks to upper case:

protected function completeCodeBlock($Block)
{
    $Block['element']['text']['text'] = strtoupper($Block['element']['text']['text']);

    return $Block;
}
