    "body": "For anyone else. I hacked up a dirty wrapper for this parser. It does two things:\r\n\r\n1. enables the possibility to have `<div>` elements:\r\n   * `[div<any number of attributes>]` to start a div container. must be separated on both ends with two newlines.\r\n   * `[/div]` to end a div container. must be separated on both ends with two newlines.\r\n2. enables `<!-- -->` HTML comments\r\n\r\nCode:\r\n\r\n~~~~\r\nfunction getMarkdown ($text) {\r\n  require_once ('/path/to/your/parsedown.php');\r\n  $result = Parsedown::instance()->parse($text);\r\n  if ((mb_strpos($result, '<p>[div') !== false) AND \r\n      (mb_strpos($result, '<p>[/div]</p>') !== false)) {\r\n    $result = preg_replace('/^\\<p\\>\\[div([^\\]]*)\\]\\<\\/p\\>/m', '<div$1>', $result);\r\n    $result = str_replace('<p>[/div]</p>', '</div>', $result);\r\n  }\r\n  if ((mb_strpos($result, '&lt;!--') !== false) AND \r\n      (mb_strpos($result, '-->') !== false)) {\r\n    $result = str_replace('&lt;!--', '<!--', $result);\r\n  }\r\n  return $result;\r\n}\r\n~~~~\r\n"
