The answer is m+Szsh => M+Szsh. 
    
The rule responsible for this is - mo'nusvAraH. (converts m->M when followed by any consonant.) 
    
Optional conversion to the fifth varga leter is done by 'anusvArasya yayi parasavarNaH'. (converts the M -> m when followed by 'yay' pratyAhAra i.e. except when followed by SzsH.)
