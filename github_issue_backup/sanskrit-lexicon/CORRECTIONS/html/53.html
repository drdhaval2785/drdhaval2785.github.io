  1. aNgakriya -> aNgakriyA

  3. atimAtra -> atimAtrA ? Similar to atidUra. I think no change required.

  4. aDoguqa -> aDoguqA

  5. tulana -> tulanA ? Similar to 2,3 above. I think no change required.
