faultfinder found spelling differences from headwords of MW, and
spelling difference was not of the 'rxx' variety, and
headword appears in sanhw1.txt list only for PD dictionary,
Since 1171 cases is a rather large number to manually check, a further programmatic examination was made.
There are many headwords among the 1171 which are compounds; if we can identify a given case as a
compound, then that case is likely to be correctly spelled.
A program was devised to catch some of these compounds; the program identifies 724 as compounds,
thus leaving 435 cases unexplained, to be examined further by reference to scanned image.
