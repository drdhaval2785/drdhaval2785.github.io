For MW, there is a fairly complete list of cases where we 'fixed' the text (i.e., made a change to mw.xml that purposely differs from the text). This list is now uploaded https://github.com/sanskrit-lexicon/MWS/blob/master/corrections-factual.txt.
There is currently no similar file for any other dictionary. Except possibly for SKD, I don't think we've thus far intentionally made changes that disagree with scans in other dictionaries than MW.
