funderburkjim commented on 2015-01-08 00:40:33

This issue looks for headword spelling errors drawn from a list of 289 alphabetical misorderings in the Wilson dictionary. In contrast to the prior list of 417 misorderings, most of these probably will not need correction. These are cases where both the misordered headwords appear in other dictionaries besides Wilson; there are 776 of these
