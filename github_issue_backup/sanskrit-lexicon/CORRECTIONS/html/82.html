This issue is for checking the spelling of headwords in the Cologne digitization for the CCS dictionary (Cappeller Sanskrit WÃ¶rterbuch) .
There are 450 cases. This issue will deal with cases 1-100, subsequent issues will be for the rest of the cases.
The headwords in these cases were identified by Dhaval's faultfinder program. Briefly, the faultfinder technique works as follows. Various patterns of character sequences that occur in the spelling of MW headwords are identified. Then analogous patterns of spelling is developed for the headwords of CCS. When a particular headword of CCS has a pattern that does NOT appear among the patters for MW headwords, then this CCS headword is identified as a POTENTIALLY misspelled headword.
For instance, the first case shown below is the CCS headword spelled 'akanizTa' .One pattern in this spelling is 'izTa' and this pattern does not occur in the spelling of any MW headword - hence, the headword is possibly misspelled in CCS.
