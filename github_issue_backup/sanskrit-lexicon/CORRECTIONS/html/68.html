funderburkjim commented on 2015-01-10 01:39:19

This begins examination of 186 headwords from Yates dictionary. These are chosen from sanhw1.txt as words that:

  1. Occur only in Yates
  2. Have some letter sequence not found in MW (this via faultfinder3a)
  3. The odd letter sequence is not of the form rXX

The examination of these cases is aided by producing suggestion words from sanhw1 that are 'nearly the same' as the questionable word, and, if possible, result in an alphabetical ordering at the position of the headword in the list of all Yates headwords.
