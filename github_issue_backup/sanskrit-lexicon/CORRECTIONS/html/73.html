funderburkjim commented on 2015-01-27 21:02:39

Yes, I think it should have been done, since 60 cases were NOT false alarms.
I could have chosen to mark the homorganic nasal spellings as NO CHANGE, and not clipped the
scans - this would have been faster, as clipping scans does take some time. If a similar situation arises in another dictionary, I may choose this faster method.
But, we need to get through the issues kicked up by faultfinder for headwords. Currently it's our best way to identify problems, with alphabetical misorderings being (for most dictionaries) a close second best way.
I'm mentioning the NO CHANGES exactly for the reason you state: to have the data needed for a NO CHANGE database of cases that we've examined, so, in some future phase of checking for headword spellings, we won't need to reconsider the ones in the database.
